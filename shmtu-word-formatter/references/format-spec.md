# 上海海事大学党政公文格式（提炼）

来源：用户提供附件《上海海事大学党政公文格式》

## 核心要求

1. 幅面：A4（210mm × 297mm）
2. 默认文字：3号仿宋（16pt）
3. 标题：2号方正小标宋简体（22pt），居中
4. 正文：3号仿宋GB_2312；每个自然段左空二字；回行顶格
5. 层级序号：
   - 一级：`一、`（黑体）
   - 二级：`（一）`（楷体）
   - 三级：`1.`（仿宋）
   - 四级：`（1）`（仿宋）
6. 日期：年/月/日使用阿拉伯数字，且月/日不补零（例如 `2026年3月5日`，而不是 `2026年03月05日`）
7. 页码：4号宋体阿拉伯数字，下方居中，推荐样式 `— 1 —`

## 自动化边界说明

脚本会自动处理：
- A4 页面
- 标题/正文/层级字体
- 首行缩进（约两字）
- 日期补零清洗
- 页脚页码（`— PAGE —`）

脚本不会自动“理解语义”处理的内容（需人工复核）：
- 标题多行时的“梯形/菱形”美观排布
- 附件说明与附件页的复杂版式
- 公章/签发人/特殊行距字距微调
